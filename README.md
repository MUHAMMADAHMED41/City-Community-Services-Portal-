# City Community Services Portal

A professional full-stack municipal management system built to modernize city services, provide a transparent complaint management system, and keep citizens informed with the latest announcements.

## Features
- **Responsive Modern UI**: Built with Bootstrap 5, glassmorphism effects, and dynamic animations.
- **Complaint Management System**: Allows citizens to easily report local issues with secure form validation.
- **Announcement System**: Keeps the public updated with the latest city notices and emergency alerts.
- **Public Services Directory**: A dynamic, searchable directory for essential municipal and emergency services.
- **Admin Dashboard**: Secure backend for city officials to track statistics, manage complaints, and publish announcements.

## Technologies
- HTML5 & CSS3
- JavaScript (Vanilla + ES6)
- PHP 8
- MySQL
- Bootstrap 5
- Font Awesome
- SweetAlert2

## Setup Instructions
1. **Import Database**: Create a database named `city_portal` in phpMyAdmin and import the `database/city_portal.sql` file.
2. **Setup Server**: Place the `city-community-portal` folder inside your local server's document root (e.g., `htdocs` for XAMPP).
3. **Start Services**: Start Apache and MySQL via your control panel.
4. **Access Portal**: Open your browser and navigate to `http://localhost/city-community-portal` (adjust path as needed).

## Admin Access
- **URL**: `http://localhost/city-community-portal/admin/login.php`
- **Default Username**: `admin`
- **Default Password**: `admin123`
